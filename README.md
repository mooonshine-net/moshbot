### Working Commands
#### Argument Usage:
* [Optional]
* "Required"

| Command | Function      | Arguments      |
|:--------|---------------|:---------------|
| shutdown| Disables the Bot [after x seconds]| [seconds]|
| addswear | add "swear" to list | "swear" |
| delswear | remove "swear" from list | "swear" |
| help    | Displays Help [about a Command] | [command]      |
